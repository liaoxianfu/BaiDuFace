## 使用说明

### 1. 账号注册 http://ai.baidu.com/

### 2 在控制台建立自己的项目 获取APP_ID API_KEY SECRET_KEY

### 3. 将BaiduFace这个文件夹导入到自己的工程目录下

### 4. 使用

####  4.1 http使用方法
不用下载sdk但是需要获取token且每个月需要动态更换（已经解决，每次自动获取token）目前只封装了人脸检测

#### 4.2 sdk使用方法
pip install baidu-aip
安装sdk后 所有封装都在face_sdk_check.py下另外testx.py为其exmple
